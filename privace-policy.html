Política de Privacidade

Esta política de privacidade foi elaborada em conformidade com a Lei Geral de Proteção de Dados (LGPD), Lei nº 13.709/2018, e descreve como coletamos, usamos e protegemos as informações fornecidas pelos usuários do nosso aplicativo.

1. Dados Coletados

Coletamos as seguintes informações pessoais:

  • Nome (não precisa ser o nome real).
  • E-mail.
  • Endereço IP (utilizado para banimentos, quando necessário).
  • Localização (coletada automaticamente junto com o IP, mas não utilizada diretamente).

2. Finalidade do Tratamento de Dados

Os dados são utilizados exclusivamente para:

  • Gerenciar e personalizar contas de usuários.
  • Garantir a segurança do aplicativo, como na aplicação de banimentos.

3. Base Legal para Tratamento dos Dados

O tratamento dos dados é realizado com base no consentimento do usuário, conforme previsto no Art. 7º, inciso I, da LGPD.

4. Armazenamento e Retenção de Dados

Os dados são mantidos até que o usuário exclua sua conta. Caso o aplicativo seja encerrado, todos os dados serão apagados permanentemente.

5. Compartilhamento de Dados

Não compartilhamos nenhuma informação coletada com terceiros.

6. Segurança das Informações

Todas as informações são armazenadas em um banco de dados protegido e mantido em sigilo. Empregamos medidas técnicas e organizacionais adequadas para proteger os dados contra acessos não autorizados.

7. Direitos do Usuário

De acordo com a LGPD, os usuários têm os seguintes direitos:

  • Confirmar a existência de tratamento de dados.
  • Acessar seus dados pessoais.
  • Corrigir dados incompletos, inexatos ou desatualizados.
  • Solicitar a exclusão de dados pessoais, caso deseje encerrar sua conta.
  • Revogar o consentimento a qualquer momento.
